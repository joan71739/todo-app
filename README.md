# Todo App

一個使用 React 製作的待辦事項管理工具，支援任務新增、編輯、刪除與狀態切換，介面簡潔、易於操作。

---

## 🚀 功能特色
- 新增、編輯、刪除待辦事項
- 標記任務完成/未完成
- 登入介面（支援帳號密碼驗證）
- 分頁導覽（首頁、登入頁等）
- 元件化設計（Navbar、Footer）
- API 服務分離（src/services/api.js）

---

## 🛠 技術棧
- React 18
- JavaScript (ES6+)
- CSS
- React Router
- Axios
